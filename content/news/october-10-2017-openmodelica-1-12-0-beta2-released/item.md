---
title: "October 10, 2017: OpenModelica 1.12.0-dev.beta2 released"
---
The OpenModelica 1.12.0-dev.beta2 has been released. You can find the Windows builds&nbsp;<a href="download/download-windows" target="_blank">here</a>. The Linux and MacOS builds are available via the Download menu above.

<strong style="color: #222222; line-height: 1.2;"><span style="font-size: 14pt;">OpenModelica 1.12.0-dev.beta2 release, October 10, 2017</span></strong>

This is a bug fix release, please check the release text for [1.12.0-dev.beta1][33] for more information.  
You can find out [which bugs were fixed][32] in the 1.12 release until now.&nbsp;

This release is in preparation of 1.12.0 final release. Please help us by testing it and [reporting bugs][4].
